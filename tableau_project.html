Airbnb Analytics Dashboard

This project leverages Tableau to analyze and visualize key trends in Airbnb listings from the 2016 dataset. The interactive dashboard explores insights into pricing patterns, property types, location trends, and host activity across various regions. By using advanced Tableau features like calculated fields, filters, and dynamic visualizations, I created an intuitive platform for stakeholders to identify market trends and opportunities. In this work I'm showing how to transform raw data into compelling stories, enabling data-driven decisions in the short-term rental industry.
